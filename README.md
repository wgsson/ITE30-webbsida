# Vibe Coding – Kort guide

Vibe Coding är en AI-genererad mjukvaruutveklingsmetod som via instuktioner genererar kod direkt istället för att skriva all manuell kod själv. Det är ett sätt att **experimentera med interaktiv kod snabbt**. Det handlar om att prova små förändringar i HTML, CSS och JavaScript och direkt se hur sidan reagerar.
